# Dane do lekcji 4 (dataset online: nycflights13)

W tej wersji Lekcji 4 pracujemy na publicznym zbiorze online `nycflights13`.

## Instalacja

```bash
pip install nycflights13
```

## Tabele używane w mini-projekcie

- `flights`
- `airlines`
- `airports`
- `weather`
- `planes`
